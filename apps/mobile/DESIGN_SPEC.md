# Karakeep Mobile App Design Specification

A beautifully designed mobile app for the Karakeep bookmarking and content curation platform.

---

## Design Philosophy

### Core Principles

1. **Content First** - Bookmarks are the hero; UI should recede and let content shine
2. **Gestural & Fluid** - Embrace mobile-native interactions (swipes, long-press, haptics)
3. **Calm Technology** - Avoid notification anxiety; be a peaceful retreat for saved content
4. **Progressive Disclosure** - Simple surface, powerful depth
5. **Offline-Ready** - Saved content should feel local and always accessible

### Visual Language

- **Color Palette**: Dark mode default with warm accents (amber/gold for favorites, soft greens for tags)
- **Typography**: Clean sans-serif (SF Pro/Inter) with generous line height for readability
- **Spacing**: Comfortable padding; content breathes
- **Motion**: Subtle, purposeful animations (spring physics, no jarring transitions)
- **Iconography**: Outlined icons, 24px touch targets, consistent stroke weight
